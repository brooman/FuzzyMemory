# Fuzzy Memory

Small memory game built in vanilla js.
You can play it here: [GH-Pages](https://laykith.github.io/FuzzyMemory)

## Local Installation

1. Clone repo: `git clone https://github.com/laykith/FuzzyMemory/`
2. Open index.html in browser
3. Done!
